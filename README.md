# Orca Tracker

A clicker game built with vanilla HTML, CSS, and JavaScript. Click to earn points, buy upgrades, unlock orca facts, and watch the background shift through colorful themes.

## How to Play

1. Open `index.html` in a browser
2. Click the **Click!** button to earn points
3. Spend points on upgrades to increase your points per click
4. Every 10 clicks, a new orca fact is revealed
5. Every 3 clicks, the background color theme changes
6. Reach **100,000 points** to win

## Features

- **Clicker Mechanic** — Earn points with each click, scaling with purchased upgrades
- **Upgrades** — Three purchasable upgrades that boost points per click
- **Orca Facts** — Learn about 8 real orca ecotypes as you play
- **Dynamic Themes** — Background and UI colors cycle through 5 themes (Ocean, Sunset, Forest, Purple, Dark) every 3 clicks with a smooth transition
- **Win Condition** — Reach 100,000 points to trigger the win screen

## Project Structure

```
index.html   - Page structure and layout
styles.css   - Base styles and 5 color theme classes
game.js      - Game logic, state management, and DOM updates
```

## Technologies

- HTML5
- CSS3 (transitions, gradients, class-based theming)
- Vanilla JavaScript (DOM manipulation, event handling)
